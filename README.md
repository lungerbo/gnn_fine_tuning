# HydraGNN GFM Fine-Tuning for Materials

Tools for fine-tuning a **single HydraGNN Graph Foundation Model (GFM)**  
(e.g., `gfm_0.229.pk`) on datasets such as **QM9**, **MD17**, and **Transition1x**.
